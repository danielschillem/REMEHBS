# Photos REMEHBS — Visages & Moments

Déposez ici les photos officielles du réseau pour qu'elles s'affichent automatiquement
sur la page d'accueil (section « Visages & moments »).

## Fichiers attendus

| Fichier                          | Sujet                              | Dimensions conseillées |
| -------------------------------- | ---------------------------------- | ---------------------- |
| `jsp-2023-pleniere.jpg`          | Plénière des Journées Scientifiques | 1200 × 800 px          |
| `formation-sages-femmes.jpg`     | Formation continue sages-femmes    | 800 × 600 px           |
| `atelier-neonatologie.jpg`       | Atelier de néonatologie            | 800 × 600 px           |
| `visite-maternite.jpg`           | Visite de maternité (CSPS)         | 800 × 600 px           |
| `recherche-equipe.jpg`           | Comité scientifique                | 800 × 600 px           |
| `ceremonie-ouverture.jpg`        | Cérémonie d'ouverture officielle   | 1200 × 600 px          |
| `familles-jsp.jpg`               | Familles & soignants               | 1200 × 600 px          |

## Format

- **JPG/JPEG** recommandé (compression web optimisée)
- **WebP** également supporté (renommer l'extension dans `HomePage.tsx` si besoin)
- Cible : 80–120 Ko par image après compression
- Outils suggérés : [Squoosh](https://squoosh.app), `cwebp`, ou ImageMagick

## Crédits & droits

Veillez à n'utiliser que des photos pour lesquelles le REMEHBS dispose d'une
autorisation écrite (consentement éclairé des personnes filmées, droits cédés
par le photographe).

## Comportement par défaut

En l'absence d'un fichier, un visuel stylisé (dégradé + pictogramme) s'affiche
automatiquement à la place : la mise en page n'est jamais cassée.
